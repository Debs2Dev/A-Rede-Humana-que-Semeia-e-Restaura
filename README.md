# A-Rede-Humana-que-Semeia-e-Restaura
Repositório focado na Rede de Sementes do Xingu e na técnica da Muvuca, integrando saberes ancestrais e inteligência artificial. O projeto detalha o impacto socioambiental de 700 coletores, apresenta um mini e-book com dados reais de restauração e documenta a engenharia de prompts aplicada na curadoria desse ecossistema de conhecimento.
